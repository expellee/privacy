Privacy Policy

Magnet Brothers (hereinafter referred to as the "Operator") establishes and discloses the following privacy policy in accordance with Article 30 of the Personal Information Protection Act to protect the personal information of data subjects and to handle related grievances promptly and smoothly.

1. Purpose of Processing Personal Information

Folders Camera (hereinafter referred to as the "App") processes the minimum amount of personal information and device permissions for the following purposes. The personal information being processed will not be used for purposes other than the following, and if the purpose of use changes, necessary measures such as obtaining separate consent will be implemented.

  • Photo and Video Recording: Providing camera functions to save photos/videos directly to a folder selected by the user.
  • File Management and Storage: Providing functions for folder creation, file moving, deletion, and album management within the device.
  • Security Services: Controlling access to specific folders via password settings and protecting privacy.

2. Items of Personal Information Processed and Permissions

The App requests the following device permissions to provide services. These permissions are essential to perform the core functions of the App.

  • Camera: Used for taking photos and videos.
  • Storage (Media File Access): Used to save captured results in specific folders and manage (move/delete) existing files.
  • Password: The password set by the user for the folder lock function is encrypted using SHA-256 and safely stored within the device. The Operator does not collect it or transmit it to a server.

3. Processing and Retention Period of Personal Information

This App does not store or collect the user's photos, videos, or password data on external servers. All data is stored only within the user's device, and if the App is deleted or if the user manually deletes the data, the Operator cannot recover it.

4. Provision of Personal Information to Third Parties

The Operator does not provide the user's personal information to third parties. All shooting data and management information are processed only within the user's device.

5. Measures to Ensure Safety of Personal Information

The Operator takes the following technical measures to protect the user's privacy.

  • Encryption: Passwords set by the user are stored after passing through a standard encryption algorithm (SHA-256), so the original password remains protected even in case of external leakage.
  • Local Processing: All photo organization and folder management processes occur offline within the device.

6. Rights and Obligations of Data Subjects and Legal Representatives and How to Exercise Them

Users can stop data processing at any time by revoking granted permissions through app settings or by deleting the App. Photos and folders stored in the device can be directly deleted at any time through the device's file management system.
